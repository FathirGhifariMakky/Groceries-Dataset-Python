## Penulisan Ilmiah

Universitas Gunadarma

---

## Deskripsi

Aplikasi ini merupakan implementasi Market Basket Analysis menggunakan Algoritma Apriori pada Groceries Dataset.

Aplikasi dibuat menggunakan Python dan Streamlit.

Pengguna dapat:

- Upload dataset
- Menjalankan Algoritma Apriori
- Menentukan Minimum Support
- Menentukan Minimum Confidence
- Melihat Frequent Itemset
- Melihat Association Rules
- Melihat Visualisasi
- Memilih barang
- Mendapatkan rekomendasi produk

---

## Dataset

Dataset yang digunakan adalah:

Groceries Dataset

Format:

```
Member_number
Date
itemDescription
```
